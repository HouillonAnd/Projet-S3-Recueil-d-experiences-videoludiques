Afin de Mettre en place le site web sur un serveur, veuillez suivre les étapes suivantes : 
1 : Glissez-déposez le répertoire contenant tous les dossiers à l'emplacement désiré sur votre serveur
2 : Allez dans le dosssier config et ouvrez Conf.php avec un éditeur de texte comme sublime text
3 : Modifiez les endroits commentés avec "//à modifier comme indiqué selon le Readme" de la façon suivante : 
	a) 'hostname' = Votre serveur
	b) 'database' = nom de la base de données
	c) 'login' = votre compte administrateur de la base de données
	d) 'password' = votre mot de passe pour la base de données.
4 : Ajoutez les tables du site web à votre base de données : copiez-collez l'intégralité du contenu du fichier bdd.sql dans une zone de texte servant à insérer des tbales ou du code sql dans votre base de données.
